# Self-RAG Mini Project

This is a simple Retrieval-Augmented Generation (RAG) mini project.

## Project Structure

- `main.py`: Main application script
- `requirements.txt`: Python dependencies
- `chroma_db/`: Local database storage
- `index.html`: (If used) Web interface

## Getting Started

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the main script:
   ```bash
   python main.py
   ```

## Description

This project demonstrates a basic RAG workflow using a local database for document retrieval and a Python backend for processing queries.
